Register

  • URL

    • /register.php
  • Method

    • POST
  • Request Body

    • name as string
    • email as string, must be unique
    • password as string, must be at least 8 characters
  • Response

      {

      "success": 1,

      "status": 201,

      "message": "You have successfully registered."

      }
