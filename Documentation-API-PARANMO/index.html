API PARANMO

Application Programming Interface(API) that will make requests and responses

  • Register
  • Login

Register

  • URL

    • /register.php
  • Method

    • POST
  • Request Body

    • name as string
    • email as string, must be unique
    • password as string, must be at least 6 characters
  • Response

      {

      "success": 1,

      "status": 201,

      "message": "You have successfully registered."

      }

Login

  • URL

    • /login.php
  • Method

    • POST
  • Request Body

    • email as string
    • password as string
  • Response

      {

      "success": 1,

      "message": "You have successfully logged in.",

      "id": "3",

      "name": "Muhammad Rohfi",

      "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."

      }
